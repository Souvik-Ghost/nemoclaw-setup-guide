# Nemoclaw Setup Guide

This repository contains a Next.js application that serves as a guide for setting up and configuring the Nemoclaw environment locally.

## Objective

This guide covers the installation and configuration of the Nemoclaw environment on a local machine (typically via WSL) to leverage local LLMs for sandbox workflows. This involves:
- Installing the Brev CLI
- Configuring the necessary environment variables and settings
- Verifying the ability to interact with the Nemoclaw agent through the terminal

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the guide.

## Modifying the Guide

You can start editing the guide by modifying `src/app/page.tsx` (or `app/page.tsx` depending on your Next.js setup). The page auto-updates as you edit the file.
